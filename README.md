# Market POS Pro PWA

سیستەمی کاشێری مارکێت — Mobile / Tablet / Desktop

ئەم وەشانە ئامادەیە بۆ دانان لە GitHub و GitHub Pages.
لە مۆبایل و تابلێت و کۆمپیوتەر کار دەکات، PWA ـە و دەتوانیت Add to Home Screen بکەیت.

## تایبەتمەندییەکان
- POS فرۆشتنی خێرا
- زیادکردنی کاڵا + بارکۆد
- کۆگا / ستۆک
- ئاگاداری کەمبوونەوەی ستۆک
- داشکاندن
- فرۆشتن بە Cash / Debt
- کڕیار و قەرز
- ڕاپۆرتی ڕۆژانە و مانگانە
- پسوڵەی 80mm و چاپکردن
- Export / Import Backup
- Offline Local Database لە ناو Browser
- ئامادە بۆ GitHub Pages

## چۆن لەسەر مۆبایل کار دەکات؟
1. GitHub repo دروست بکە
2. هەموو فایلەکان Upload بکە
3. Settings > Pages
4. Deploy from branch هەڵبژێرە
5. Branch: main / root
6. لینکەکەت بکەرەوە لە Chrome
7. Add to Home Screen بکە
